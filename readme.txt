This is chat app for Android that is based on googles firebase infrastructure. Backend is written in PHP and it is using MySql database. This app can transport text and images to other user of the same app in realtime.
List of some used libraries:
- BroadcastReceiver
- Service
- CardView
- RecycleView
- Volley
- Toolbar
- ViewPager
- firebase-messaging
